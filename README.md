# FarmEasy - Ecommerce Website

FarmEasy is a simple ecommerce website for agricultural and farming products. It is built using HTML, CSS, and JavaScript.
## Features

- Browse a wide range of agricultural products.
- Add products to the cart.
- View product details with images and descriptions.
- Secure checkout process.
- Responsive design for mobile and desktop.
